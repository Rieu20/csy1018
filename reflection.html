My overall experience was intimidating as a beginner learning coding languages in web development module using HTML5 and CSS3. I felt very uncomfortable and lost during the first 3 weeks. However, with the help, support and resources available on NILE, combined with support sessions from (Teaching Assistant Andrew and Module tutor Liz) I was able develop and improve my coding skill.

Another helpful approach I took upon my self was making sure I watch the tutorial video’s uploaded by my module tutor weekly. My project was to build a current ‘User Interface design’ based on current design trends as exhibited on Google Material Design approach creating 6 web pages with technical scripting skills such as the designing, usability, navigations and testing the code.

After an extensive research on how to start the project, together with help and support it all started fallen into place. I eventually created my six web pages using the syntax of HTML and CSS with a new updated trending application called THIMBLE by MOZILLA https://thimble.mozilla.org/en-GB/ thimble.mozilla Thimble is an online code editor I found very effective and easy to use when creating your personal web page, as its accessible to view, edit, and check your projects at any time or place once connected through the net.
